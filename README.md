## Стек

- **Frontend:** Vanilla JS (ES modules), Parcel 2
- **Backend:** Node.js + Express
- **Хранилище:** localStorage (корзина, избранное, сравнение)
- **Шрифт:** Maven Pro (Google Fonts)

## Запуск

```bash
# Backend (порт 3000)
cd dev-server
node src/index.js

# Frontend (порт 1234)
cd vercel-client
npm install
npm start
```

Открыть: `http://localhost:1234`

## Значимые изменения относительно оригинала

### Дизайн

- **Полный редизайн в стиле светлого минимализма** — тёплый кремовый фон (`#F5F0EB`), белые карточки, бордово-винный акцент (`#8B2635`) вместо тёмной тематики
- **Сетка карточек** — переход с 2 на 3 колонки, карточки стали компактнее и соразмернее изображению бутылки
- **Иерархия кнопок в карточке** — «В корзину» — главная акцентная кнопка (бордовая), «Подробнее» — второстепенная (обведённая)
- **Шапка** — фон растянут на всю ширину экрана; иконка бутылки NaVino совпадает по цвету с логотипом (`fill="currentColor"`)
- **Блок фильтров** — убраны тени, кнопки автоматически подстраиваются под ширину текста, элементы управления не выходят за границы блока
- **Навигационные вкладки** — заменены крупные `<h4>` (20px) на компактный uppercase 11px

### Функциональность

- **Рабочие фильтры** — тип, цвет, содержание сахара, диапазон цен фильтруют каталог через shared state (`state.js`) и кастомное событие `navino:filters-changed`
- **Поиск в реальном времени** — поле в шапке диспатчит `navino:search-changed`, каталог мгновенно фильтруется
- **Корзина** — добавление товаров, счётчик-бейдж в шапке, хранение в `localStorage`
- **Избранное** — кнопка на карточке, счётчик в шапке, состояние сохраняется между сессиями
- **Сравнение вин** — до 3 вин одновременно, новая страница `comparison.html` с таблицей характеристик
- **Исправлена API** — оригинальный Vercel-эндпоинт возвращал 404; переключено на локальный Express-сервер

### Новые разделы навигации

| Раздел | Содержимое |
|--------|-----------|
| **Персональные подборки** | Личный кабинет: профиль, статистика (заказы / избранное / отзывы), рекомендации по рейтингу ≥ 4.0, история заказов. Требует авторизации |
| **Выгодные предложения** | Каталог с фильтрами и поиском |
| **Дегустации** | 6 карточек мероприятий с датой, местом, ценой и кнопкой записи |
| **Обучение** | 6 курсов трёх уровней: Начинающий / Средний / Продвинутый |

### Навигация между страницами

- Клик по разделу со страницы товара переходит на `index.html?section=X` — секция активируется автоматически по URL-параметру
